## SquareIt

A very simple and moddable game, written in Python using pygame.

> It's in a very early stage, so there's not much to play right now ;)

<br>

## Play

Clone the repo, cd into the cloned folder and into the `Source` folder.
Then you just run:
`python Run.py`

<br>

## Requirements

SquareIt uses:
- json
- pygame

<br>
